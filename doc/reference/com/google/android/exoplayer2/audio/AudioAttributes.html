Class AudioAttributes

  • java.lang.Object
    • com.google.android.exoplayer2.audio.AudioAttributes

  • public final class AudioAttributes
extends Object
    Attributes for audio playback, which configure the underlying platform AudioTrack.

    To set the audio attributes, create an instance using the AudioAttributes.Builder and either pass it to SimpleExoPlayer.setAudioAttributes(AudioAttributes) or send a message of type C.MSG_SET_AUDIO_ATTRIBUTES to the audio renderers.

    This class is based on AudioAttributes, but can be used on all supported API versions.
