Class ChunkExtractorWrapper

    public final class ChunkExtractorWrapper
extends Object
implements ExtractorOutput
    An Extractor wrapper for loading chunks that contain a single primary track, and possibly additional embedded tracks.

    The wrapper allows switching of the TrackOutputs that receive parsed data.
