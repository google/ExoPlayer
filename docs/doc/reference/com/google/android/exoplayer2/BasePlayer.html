      • seekToPrevious

        public final void seekToPrevious()
        Description copied from interface: Player
        Seeks to an earlier position in the current or previous MediaItem (if available). More precisely:
        • If the timeline is empty or seeking is not possible, does nothing.
        • Otherwise, if the current MediaItem is Player.isCurrentMediaItemLive() live} and unseekable, then:
          • If a previous media item exists, seeks to the default position of the previous media item.
          • Otherwise, does nothing.
        • Otherwise, if a previous media item exists and the current position is less than Player.getMaxSeekToPreviousPosition(), seeks to the default position of the previous MediaItem.
        • Otherwise, seeks to 0 in the current MediaItem.

        This method must only be called if Player.COMMAND_SEEK_TO_PREVIOUS is available.
