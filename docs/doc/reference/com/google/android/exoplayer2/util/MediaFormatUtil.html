      • KEY_EXO_PCM_ENCODING

        public static final String KEY_EXO_PCM_ENCODING
        Custom MediaFormat key associated with an integer representing the PCM encoding.

        Equivalent to MediaFormat.KEY_PCM_ENCODING, except it allows additional ExoPlayer-specific values including C.ENCODING_PCM_16BIT_BIG_ENDIAN, C.ENCODING_PCM_24BIT, and C.ENCODING_PCM_32BIT.
