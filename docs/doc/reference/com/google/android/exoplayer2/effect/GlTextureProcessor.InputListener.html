Interface GlTextureProcessor.InputListener

  • Enclosing interface:
    GlTextureProcessor
    public static interface GlTextureProcessor.InputListener
    Listener for input-related frame processing events.

    This listener can be called from any thread.

        Called when the GlTextureProcessor is ready to accept another input frame.

        default void onInputFrameProcessed​(TextureInfo inputTexture)
        Called when the GlTextureProcessor has processed an input frame.

        The implementation shall not assume the GlTextureProcessor is ready to accept another input frame when this method is called.
