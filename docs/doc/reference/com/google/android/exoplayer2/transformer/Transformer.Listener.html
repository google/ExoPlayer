Package com.google.android.exoplayer2.transformer

Interface Transformer.Listener

  • Enclosing class:
    Transformer
    public static interface Transformer.Listener
    A listener for the export events.

    If the export is not cancelled, either onError(com.google.android.exoplayer2.transformer.Composition, com.google.android.exoplayer2.transformer.ExportResult, com.google.android.exoplayer2.transformer.ExportException) or onCompleted(com.google.android.exoplayer2.transformer.Composition, com.google.android.exoplayer2.transformer.ExportResult) will be called once for each export.
