Annotation Type MediaMetadata.MediaType

  • @Documented
@Retention(SOURCE)
@Target(TYPE_USE)
public static @interface MediaMetadata.MediaType
    The type of content described by the media item.

    One of MediaMetadata.MEDIA_TYPE_MIXED, MediaMetadata.MEDIA_TYPE_MUSIC, MediaMetadata.MEDIA_TYPE_AUDIO_BOOK_CHAPTER, MediaMetadata.MEDIA_TYPE_PODCAST_EPISODE, MediaMetadata.MEDIA_TYPE_RADIO_STATION, MediaMetadata.MEDIA_TYPE_NEWS, MediaMetadata.MEDIA_TYPE_VIDEO, MediaMetadata.MEDIA_TYPE_TRAILER, MediaMetadata.MEDIA_TYPE_MOVIE, MediaMetadata.MEDIA_TYPE_TV_SHOW, MediaMetadata.MEDIA_TYPE_ALBUM, MediaMetadata.MEDIA_TYPE_ARTIST, MediaMetadata.MEDIA_TYPE_GENRE, MediaMetadata.MEDIA_TYPE_PLAYLIST, MediaMetadata.MEDIA_TYPE_YEAR, MediaMetadata.MEDIA_TYPE_AUDIO_BOOK, MediaMetadata.MEDIA_TYPE_PODCAST, MediaMetadata.MEDIA_TYPE_TV_CHANNEL, MediaMetadata.MEDIA_TYPE_TV_SERIES, MediaMetadata.MEDIA_TYPE_TV_SEASON, MediaMetadata.MEDIA_TYPE_FOLDER_MIXED, MediaMetadata.MEDIA_TYPE_FOLDER_ALBUMS, MediaMetadata.MEDIA_TYPE_FOLDER_ARTISTS, MediaMetadata.MEDIA_TYPE_FOLDER_GENRES, MediaMetadata.MEDIA_TYPE_FOLDER_PLAYLISTS, MediaMetadata.MEDIA_TYPE_FOLDER_YEARS, MediaMetadata.MEDIA_TYPE_FOLDER_AUDIO_BOOKS, MediaMetadata.MEDIA_TYPE_FOLDER_PODCASTS, MediaMetadata.MEDIA_TYPE_FOLDER_TV_CHANNELS, MediaMetadata.MEDIA_TYPE_FOLDER_TV_SERIES, MediaMetadata.MEDIA_TYPE_FOLDER_TV_SHOWS, MediaMetadata.MEDIA_TYPE_FOLDER_RADIO_STATIONS, MediaMetadata.MEDIA_TYPE_FOLDER_NEWS, MediaMetadata.MEDIA_TYPE_FOLDER_VIDEOS, MediaMetadata.MEDIA_TYPE_FOLDER_TRAILERS or MediaMetadata.MEDIA_TYPE_FOLDER_MOVIES.
